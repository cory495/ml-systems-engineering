# Data Models

Date: 2026-06-07

---

## 1. Definition

---
A data model is a blueprint on how data is collected, stored, and processed in a system.
## 2. Why It Matters

---
Data models are important because they affect how we think about the problem we are solving.
## 3. Key Ideas

---
- Data models encode real-world things, and APIs manipulated said data models.
- JSON and XML documents, relational tables, and graph models are general-purpose data models.
## 4. Examples

---
- People can be encoded as data models.
- Organizations can be encoded as data models.
- Goods can be encoded as data models.
- Actions can be encoded as data models.
- Money flows can be encoded as data models.
- Sensors can be encoded as data models.
