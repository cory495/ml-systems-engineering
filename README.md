# ML Systems Engineering Knowledge Base

A personal knowledge management system built in Obsidian to support graduate study, research, interview preparation, and machine learning systems engineering.

## Purpose

This repository serves as a long-term learning and research platform focused on:

* Machine Learning Systems
* Distributed Systems
* Data Engineering
* Computer Architecture
* Operating Systems
* Networking
* Databases
* Software Engineering
* System Design
* Applied Mathematics

The goal is to develop the depth of knowledge required for advanced ML infrastructure, distributed systems, and research engineering roles.
